# v26 会议包深度质询与修正方案 (Critical Review & Action Plan)

**审查目标：** 剥离过度工程防卫，直击核心科学逻辑漏洞，确保答辩与盲审时的逻辑无懈可击。

------

## 一、 四大致命逻辑漏洞与修正建议

### 漏洞 1：消融数据与核心 Claim 发生严重冲突

- **现状（被攻击点）：** 你们在 Q&A (Q3) 中声称收益来源于“Mutual NN 匹配所带来的高质量运动场”。然而，在“3.5 机制证据”中，`planb_ablate_no_mutual_smoke200` 相比 Default 仅退化了 **ΔPSNR -0.0534**。
- **评委视角：** “Plan-B 总体提升了 1.5 dB，但你把所谓的‘核心’ Mutual NN 拿掉，只掉了 0.05 dB。这说明剩下的 1.45 dB 纯粹是靠打破‘零速’带来的！你凭什么说核心贡献是 Mutual NN？”
- **修正方案（立即修改叙事）：** * **不要过度包装 Mutual NN。** 诚实地承认客观事实：在稀疏视角 4DGS 中，“零速初始化”是一个巨大的收敛陷阱。
  - **重塑 Claim：** Plan-B 的核心贡献是**“引入物理运动先验打破零速陷阱”**（贡献了主要 PSNR 增益），而 Mutual NN 等组件的作用是**“时序去噪与轨迹平滑（稳定 tLPIPS）”**。

### 漏洞 2：绝对指标过低（PSNR 20.4）的“原罪”

- **现状（被攻击点）：** 从 18.9 涨到 20.4 是巨大的相对提升（+1.5dB），但在 2026 年的 4D 重建标准下，20.4 的绝对 PSNR 依然会被视为“未收敛”或“全损画质”。
- **评委视角：** “你花了大量篇幅证明没有 cherry-pick，但这渲染出来的视频依然很糊。这到底是算法不行，还是你们的超参 `max_steps_full=600` 把模型卡死了？”
- **修正方案（严格限定 Scope）：**
  - 在会议和论文中，**主动弱化“高保真（High-Fidelity）”这一标签**。
  - 将研究范围严格框定为：**“极短训练预算（600 steps）与稀疏视角下，4DGS 初始化策略的收敛性与时序稳定性分析”**。卖点是“收敛速度”和“跳出局部最优”，而非最终绝对画质。

### 漏洞 3：Feature Loss (v2) 与 Plan-B 的虚假对立

- **现状（被攻击点）：** 你们把 Feature Loss 判了死刑（No-Go），把 Plan-B 捧上位（Go）。
- **评委视角：** “Feature Loss 是一种**训练期正则化损失（Regularization）**，Plan-B 是一种**初始化策略（Initialization）**，两者正交。Feature Loss 失败很可能是因为它被困在劣质的‘零速基底’里。如果用 Plan-B 给出良好初速度再加 Feature Loss 呢？你没做这个实验，凭什么彻底否定 Feature Loss？”
- **修正方案（转化为 Limitations）：**
  - **在论文中主动声明局限性：** “在零速初始化的劣质基底上，纯 2D 语义特征约束（Feature Loss）无法单骑救主，甚至会产生优化对抗。受限于算力预算，本研究未测试良性初始化下的特征约束，但实证了物理初始化（3D 运动先验）具有更高的优先级和有效性。”

### 漏洞 4：会议议程过于“防守”，缺乏视觉“进攻”

- **现状（被攻击点）：** 议程安排了大量时间讨论 SHA256 校验和 Template Hygiene。
- **评委视角：** 导师默认你的工程是严谨的。花大量时间自证清白（讲反作弊机制），反而显得对核心结果心虚。
- **修正方案（调整会议焦点）：**
  - 将“防 cherry-pick”机制一句话带过（“已通过严苛的多段盲测交叉验证”）。
  - **主攻定性视频（Side-by-side）：** 把省下的时间用于播放对比视频。让导师直观看到 Baseline 的重影/闪烁是如何被 Plan-B 修正的。**视觉冲击力是对抗低 PSNR 质疑的唯一武器**。

## 二、 会议战术与话术升级

### 1. 砍掉选项，强势收口 (Close the Deal)

不要在会议上给导师“继续做实验”的选择题（即砍掉原文档中的选项 B/C）。

- **话术：** “基于 v26 极其坚固的证据链（Canonical + Seg200 + 4 段 Smoke 交叉验证），我们在实证层面已无盲区。**我强烈建议直接执行选项 A（代码冻结，全员转入写作）**。任何新增的短期实验边际收益均为零，且会破坏当前已锁定的 Evidence Pack。”

### 2. 核心 Q&A 话术替换（针对 Q3 投机质疑）

- **原版 Q3：** Plan-B 是不是靠“速度从 0 变成非 0”投机？
- **升级版回答：** “打破零速陷阱确实是我们获得 1.5dB 提升的核心物理机制，但这绝不是简单的‘非零投机’。我们通过消融实验（如 `planb_ablate_no_mutual`）证明，如果不引入严谨的 Mutual NN 匹配约束，单纯赋予初速度无法实现如此平滑的时序稳定性。此外，能在 6 个完全不同的时间切片上稳定实现 tLPIPS 的显著下降，充分证明了该 3D 物理运动场先验的普适性与鲁棒性。”

### 3. 叙事升华 (The Storyline)

在最终论文中，采用以下“破局”叙事结构：

1. **痛点发现：** 稀疏视角 4DGS 极易陷入几何崩溃与时序闪烁。
2. **反直觉挫折（v2 失败）：** 引入强大的 2D 大模型特征（VGGT）试图兜底，却发现 2D 特征缺乏多视角几何一致性，反而与 3DGS 产生优化对抗（PSNR 下降）。
3. **回归物理（Plan-B 破局）：** 放弃复杂的语义外挂，回归第一性原理。通过三角化点云提供粗糙但符合物理直觉的 3D 速度先验，以极低成本实现了极大的收敛飞跃。
