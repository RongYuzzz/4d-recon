## 一、 开题报告（4D-Reconstruction.md）精准修订指南

请对照同行评审意见，在原开题报告中进行以下修改，以确保答辩时的逻辑无懈可击：

### 1. 明确创新点与对标差异（回应意见 2.1）

在“主要内容与预期目标”部分前，新增**【本文主要贡献】**段落：

* **对比 FreeTimeGS：** 突破其零速初始化的局限，引入显式的 3D 物理运动先验与大模型几何语义约束，显著提升稀疏视角下的时序稳定性（tLPIPS）。
* **对比 Split4D：** 抛弃其依赖复杂逐帧对比学习的重度管线，提出一种更轻量的、基于特征空间渲染比较（Render-and-Compare）的联合优化策略。
* **对比 VGGT4D：** 避免仅将大模型作为 2D 掩码生成器，而是深入到 3D 高斯特征场层面进行时空约束。

### 2. 修正“端到端”表述漏洞（回应意见 2.2）

* **修改动作：** 全局搜索“不依赖任何外部2D分割或跟踪器作为前置模块”。
* **替换为：** “**无需外部 2D 强监督（如 SAM/DEVA 预处理掩码）的联合优化范式**”。
* **说明：** 明确 VGGT 提取的特征和伪掩码属于“无监督/自监督的软先验（Soft Prior）”，以 Loss 形式在训练中联合优化，而非硬性切断流水线。

### 3. 落地“注意力引导对比损失”的数学定义（回应意见 2.3）

专家指出像素级注意力复杂度为 $O((HW)^2)$ 无法实现。需在方法部分补充具体的降维与计算策略：

* **特征绑定：** 为每个 4D 高斯附加一个低维可学习特征 $f_i$，通过 Splatting 渲染为 2D 语义特征图 $F_{render}$。
* **稀疏注意力构建：** 提取 VGGT 在 $t$ 和 $t+k$ 帧的特征图，通过 **Patch-level（区块级）或 Top-K 关键点匹配** 构建稀疏对应关系矩阵 $A$。
* **损失定义：** 若注意力矩阵表明 $t$ 帧像素 $p$ 与 $t+k$ 帧像素 $q$ 对应，则最小化其渲染特征距离：

$$L_{feat} = || F_{render}^t(p) - F_{render}^{t+k}(q) ||_2$$



### 4. 调整评测口径与目标（回应意见 2.4）

* **修改动作：** 删除开题中关于“Neural3DV、Multi-Human”以及“mIoU 保证优于...”的不切实际的承诺。
* **替换为：** “在稀疏视角动态数据集（如 SelfCap）和极短训练预算下，重建的时序稳定性指标（**tLPIPS**）显著优于 Baseline，定性上大幅消除重影（Ghosting）和闪烁（Flicker）现象。”

### 5. 补充“资源约束与止损策略”章节（回应意见 2.5）

* **新增小节：** 在“总体安排”后加入资源说明。明确单次全量实验（600 steps）的算力开销，并说明若高级语义约束（如 Feature Loss）未达预期，将以**物理速度初始化（Plan-B）**作为核心保底交付物。
